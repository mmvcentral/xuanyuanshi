## Character Declaration Files

### 声明.txt (Declaration by MASTCHEN)

| Original (Chinese) | English Translation |
|-------------------|---------------------|
| MASTCHEN关于轩辕诗的几点申明 | MASTCHEN's Declaration Regarding Xuanyuan Shi |
| 我在制作这个角色时的初衷就是创作一个"百分之百的中国角色" | My original intention when creating this character was to create a "100% Chinese character" |
| 虽然形象上借鉴了《怒之铁拳》的BLAZE | Although the appearance references BLAZE from "Streets of Rage" |
| 但阿诗是一个从头到尾都百分百的中国妹子 | A-Shi is 100% Chinese from head to toe |
| 之所以像BLAZE，是因为她小时候玩游戏时崇拜上了这个角色 | The reason she resembles BLAZE is because she admired this character when playing games as a child |
| 因此，阿诗的配音也只承认中文 | Therefore, A-Shi's voice acting only acknowledges Chinese |
| 在此对给予了阿诗声音的冬弓小姐表示万分感激 | I express my heartfelt gratitude to Miss Donggong for providing A-Shi's voice |
| 她配出了我想象中阿诗的声音 | She voiced A-Shi's voice as I imagined it |
| 希望百分百国产的轩辕诗，能够成为国产mugen引以为傲的角色 | I hope the 100% domestically-made Xuanyuan Shi can become a character that domestic MUGEN can be proud of |
| 11区所发布的"轩辕诗"并非轩辕诗本人 | The "Xuanyuan Shi" released in District 11 is not Xuanyuan Shi herself |
| 而是轩辕诗的cosplay | But rather a cosplay of Xuanyuan Shi |
| 在人设上是一对爱好cosplay的姐妹，分别是姐姐"示"和妹妹"紫" | In character design, it's a pair of cosplay-loving sisters: older sister "Shi" and younger sister "Zi" |
| Fkof中的挑战者轩辕诗，人设为一个大会幕后组织利用轩辕诗细胞所克隆的克隆体，仅拥有阿诗的部分能力 | The challenger Xuanyuan Shi in Fkof is designed as a clone created by a tournament幕后 organization using Xuanyuan Shi's cells, possessing only partial abilities of A-Shi |
