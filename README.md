# A-Shi (轩辕诗 / Xuanyuan Shi)

A MUGEN character by **mast-chen** (mastchen). Version: 02/03/2005. Compatible with MUGEN 1.0.

---

## Character Introduction

**A-Shi** (Xuanyuan Shi / 轩辕诗) is a Chinese high school girl who enjoys travel, fighting, and new experiences. She travels and fights in the MUGEN world out of interest and a sense of justice.

### Background

- **Origin**: A martial arts family with a secret sealed for thousands of years
- **Bloodline**: Huangdi (Yellow Emperor) lineage
- **Abilities**:
  - Thunder Dragon: from Huangdi's general "Ying Long" (应龙)
  - Hellfire: from Huangdi's daughter "Nü Ba" (女魃)

## System Architecture

### File Structure

```
xuanyuanshi/
├── xuanyuanshi.def    # Character definition (load this in MUGEN)
├── A-Shi.cns          # Main CNS (large)
├── A-Shi.air          # Animation definitions
├── A-Shi_Normal_Special.cns   # Normal/Special states
├── A-Shi_Super.cns    # Super moves
├── A-Shi_common-2.cns # Common states
├── hit.cns            # Hit detection
├── common-3.cns       # Common systems (AI, command buffer)
├── power_full_max.cns # Power/MAX states
├── command.cmd        # Command definitions
├── data.txt           # Data overrides
├── show_menu.air      # Menu animation definitions
├── stroy.def          # Arcade intro
├── ending.def         # Arcade ending
├── command list/      # Command list (HTML)
├── No H Patch/        # Non-H variant
├── 16X Patch/         # 16X variant
├── 18X Patch/         # 18X variant
└── docs/              # Documentation
```

*Note: The character loads `xuanyuanshi.def`; the def references `A-Shi.sff`, `A-Shi.snd`, and `A-Shi.air` for sprites, sound, and animations.*

### CNS Module Roles

| File | Purpose |
|------|---------|
| `A-Shi_Normal_Special.cns` | Base states, movement, normal/special moves |
| `A-Shi_Super.cns` | Dragon series supers, special moves |
| `A-Shi_common-2.cns` | Reversal defense, crouching normals, AI |
| `hit.cns` | Hit detection, hit states |
| `common-3.cns` | Global AI, command buffer, state detection |
| `power_full_max.cns` | Power release, Divine Battle, Forbidden Secret |

### Fight Mode Architecture

- **State types**: S (Standing), C (Crouching), A (Air)
- **Physics**: S (Stand), C (Crouch), N (None), A (Air)
- **fvar(10)**: Mode switch (0=Normal/Qinglong, 1=Fengshen)
- **var(59)**: AI enable flag
- **var(57)**: AI difficulty (1–11)

### Skill Categories

1. **Dragon Series** (青龙): Thunder-based moves
2. **Phoenix Series** (赤凤): Flame-based moves
3. **Demon Dragon** (魔龙): Steel fist, bite, rush
4. **Super Moves**: 百龙之神, 噬魂之炎, 龙拳爆发, etc.

### Animation Mapping

- **Standing normals**: 228, 229 (crouching: 428, 438)
- **Super / special**: 3000–4999 range
- **Divine Battle / Power Release**: 80000+ range
- See `docs/TRANSLATION.md` for full skill-to-animation mapping.

### Counter Parties

- **ReversalDef**: Reversal defense for certain states
- **HitOverride**: Armor override for specific moves
- **p2stateno**: Counter-hit states (5000, 5010, 5020, etc.)
